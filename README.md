## Termux setup
> Install packages
```bash
apt install termux-api python neovim git nodejs-lts rigrep ruff luarocks lua-language-server 
```
> Setup nvim
```bash
mv ~/.config/nvim/ ~/.config/nvim.bak
git clone https://github.com/Veha0001/VimLazyUser ~/.config/nvim
```
